# HOUSE PRICE PREDICTION PROJECT

A machine learning model to predict house prices based on various features. This project was built in Google Colab using a dataset from Kaggle.

---

## About The Project
This project is an machine learning task. The goal was to pratice data cleaning, model building and find out best model. The final model is Random Forest Regressor that predicts house sale prices.

---

## Dataset
The dataset was sourced from **Kaggle**.
You can find and download the dataset here: (https://www.kaggle.com/datasets/shree1992/housedata)
*Due to its size, the data is not included in this repository.*
